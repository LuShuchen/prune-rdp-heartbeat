# Privacy Policy

**Last Updated:** February 5, 2026

This Privacy Policy describes how **RDP Heartbeat** ("the Application") collects, uses, and discloses your information.

## 1. Data Collection
The Application **does not collect, store, transmit, or share any personal information**, usage data, or telemetry.

The Application operates entirely locally on your device. All settings and preferences (such as dot color, size, and position) are stored locally in a configuration file on your computer and are not transmitted to us or any third parties.

## 2. Internet Access
The Application does not connect to the internet, with the exception of:
- Opening the project's GitHub repository page in your default web browser if you explicitly click the "Visit Project Website" link in the "About" dialog.

## 3. Third-Party Services
The Application does not use any third-party analytics, advertising services, or tracking SDKs.

## 4. Contact Us
If you have any questions about this Privacy Policy, please contact us via the project's GitHub repository:
https://github.com/LuShuchen/prune-rdp-heartbeat
