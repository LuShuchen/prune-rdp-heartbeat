## 2. 商店列表信息 (Store Listings)

### 描述 (Description)
*注意：直接复制以下内容到商店后台的 Description 框中。*

远程桌面到底卡没卡？一眼便知。

RDP Heartbeat 是一款极简的“远程连接保活”工具。它会在屏幕上生成一个微小的呼吸光点：
• 光点在跳动：连接正常，画面依然在传输。
• 光点停止了：你的 RDP 连接已经卡死或断开。

跑代码、炼模型或者长时间编译时，屏幕画面静止是常态。究竟是程序在思考，还是网络已经掉线？RDP Heartbeat 能立刻给你答案，让你告别“假死焦虑”。

核心功能：

• 可视化心跳：一个平滑呼吸的动态光点，直观确认远程画面是否“活着”。

• 开机自启：支持随系统自动启动，一次设置，永久省心（v1.1 新增）。

• 无感体验：“总在最前”确保随时可见，同时支持“鼠标穿透”模式，点击光点位置时操作会直接穿透到下方窗口，完全不影响工作。

• 高度客制化：
   - 颜色：随心调整，匹配你的壁纸风格。
   - 大小与透明度：从几乎隐形的像素点到清晰的指示灯，由你定义。
   - 位置：拖拽即可放置在屏幕任意角落。

• 原生中文支持：界面已完全本地化，支持简体中文。

• 纯净轻量：安静地最小化到系统托盘，占用极低资源。

专为远程办公和极客打造，让远程连接更稳定、更省心。
